# JSON HTTP Request Tag for Google Tag Manager Server Side

The **JSON HTTP Request Tag** is a custom tag for the Google Tag Manager server container that allows you to send event data to any third-party endpoint via an HTTP request with a JSON payload.

This tag is highly flexible, enabling you to forward data from your server-side GTM to CRMs, analytics platforms, or any other service that accepts JSON data.

## Useful Resources
- How to use it and more info can be found in [this article](https://stape.io/how-to-send-all-ga-event-data-to-your-url-site-crm/).
